What’s Inside Each 30-Day Program

Structured Blocks

6 blocks × 5 days each. Every block has drills, QR-linked demos, and a Mark Completed button to track progress.

  • Warm-up & activation
  • Core skill drills
  • Cool-down & reflection

Real Skill Transfer

Progress from fundamentals → consistency → game application. Build decision-making, not just isolated reps.

  • Technique cues
  • Constraints for realism
  • Self-checks

Track & Improve

Checklists + progress bars and (on program pages) quick notes you can save for reminders and reflections.

  • Session completion
  • Personal notes (saved locally)
  • Weekly prompts

How to Use This Site

1) Pick Your Level

Start where you can succeed today. Move up when you complete most blocks with good form.

2) Scan & Copy

Open the QR on each block to watch the demo. Copy the technique cues and set a small goal (e.g., “10 clean makes”).

3) Track Progress

Hit Mark Completed after each block. Use the notes area (on program pages) to jot quick reflections.

Weekly Rhythm

Mon

Handles + Layups

  • Use glass on finishes

Shooting (Form → Rhythm)

45–60 min

  • B.E.E.F. checkpoints
  • Consistent routine
Fri

Defense & Conditioning

45–60 min

  • Quiet feet, angles, hands
  • Short, sharp intervals
Sat

Small-Sided Games

45–60 min
